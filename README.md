# Mandelbrot Set Visualizer
This is a simple program that allows the user to visualize the mandelbrot set. There are three main variables that can be modified. 
- The number of divisions <br>
When the number of divisions is increased so to is the size of the plot that is generated. Divisions by default are set to 1000 by the variable "div". <br>
- The number of loops <br>
Increasing the number of loops increases the resolution of the resultant plot as this makes the calculation more accurate. By default this is set to 100 under the "loop" variable. <br>
- The color scheme of the plot <br>
The color scheme of the plot can also be modified through the "colour" variable. The available colors are accessible from the pyplot docs [here](https://matplotlib.org/stable/gallery/color/colormap_reference.html).
